Title: Organization
----

Text:

### List

Required permission:

* ticket.agent or admin.organization (can read all organizations)
* any (can only read it's own organization if exists)

Request:

```bash
GET /api/v1/organizations
```

Response:

```bash
Status: 200 Ok

[
  {
    "id": 123,
    "name":"Org 1",
    "shared": true,
    "active": true,
    "note":"some note",
    "updated_at": "2016-08-16T07:55:42.119Z",
    "created_at": "2016-08-16T07:55:42.119Z"
  },
  {
    "id": 124,
    "name":"Org 2",
    "shared": false,
    "active": true,
    "note":"some note",
    "updated_at": "2016-08-16T07:55:42.119Z",
    "created_at": "2016-08-16T07:55:42.119Z"
  },
]

```

### Show

Required permission:

* ticket.agent or admin.organization (can read all organizations)
* any (can only read it's own user if exists)

Request:

```bash
GET /api/v1/organizations/{id}
```

Response:

```bash
Status: 200 Ok

{
  "id": 123,
  "name":"Org 1",
```

### Create

Required permission:

* admin.organization

Request:

```bash

  "shared": true,
  "active": true,
  "note": "some note"
}
```
